# Implementasi Frontend

## Kaitan Data table.
1. semua action di table data ada di sisi kanan.
2. aksi berupa tombol icon dengan menggunakan `chakra-ui`. 
3. jika aksi terlalu banyak maka gunakan menu. untuk referensi [Lihat Ini](https://chakra-ui.com/docs/components/menu).
4. aksi di menu item punya ikon di sisi kiri.
