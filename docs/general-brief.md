
# Fitur Besar

## 1. Beranda

Beranda menyediakan ringkasan statistik pembelajaran secara real-time, meliputi:
- Total kelas
- Total siswa
- Total materi
- Total tugas
- Jumlah tugas yang telah dikumpulkan
- Jumlah tugas yang belum dikumpulkan
- Jumlah tugas yang perlu dinilai
- Rata-rata nilai siswa
- Statistik siswa per jurusan
- Statistik siswa per kelas
- Status materi (Draft & Dipublikasikan)
- Daftar tugas terbaru
- Daftar pengumpulan tugas terbaru

## 2. Mata Pelajaran

### 1. Manajemen Daftar mapel
- Membuat mata pelajaran.
- Mengubah data mata pelajaran.
- Menghapus mata pelajaran.
- Mengelompokkan mata pelajaran berdasarkan kelas.

### 2. Manajemen Materi Umum
- Membuat materi pembelajaran.
- Upload file materi.
- Draft dan Publish materi.
- Edit dan hapus materi.

### 3. Manajemen Tugas
- Membuat tugas.
- Menentukan batas waktu pengumpulan.
- Mengelola daftar tugas.
- Monitoring pengumpulan tugas.

### 4. Pengumpulan Tugas
- Siswa mengumpulkan tugas.
- Guru melihat hasil pengumpulan.
- Status sudah/belum dikumpulkan.

### 5. Penilaian
- Memberikan nilai.
- Memberikan komentar.
- Rekap nilai siswa.
- Riwayat penilaian.

### 6. Manajemen Log Aktifitas 
- Riwayat aktivitas pengguna.
- Monitoring aktivitas guru.
